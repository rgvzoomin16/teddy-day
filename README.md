# TeddyVerse

An interactive Teddy Day web experience that demonstrates advanced DSA ideas in a playful format.

## Included feature mapping

- Graph + DFS/BFS: branching love story with shortest path to the best ending.
- Topological sort (Kahn): prerequisite-based surprise unlocks.
- Trie: prefix suggestions for quote search.
- Dynamic Programming: typo correction using Levenshtein distance.
- Grid BFS: teddy maze shortest path solver.
- Heap: top-k leaderboard (fastest times).
- Hash indexing: mood-based assistant quote retrieval.
- Fisherâ€“Yates shuffle: non-repeating surprise randomizer.
- Bitmasking: achievement progress and unlock.
